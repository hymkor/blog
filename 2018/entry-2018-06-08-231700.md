```header
Title: バッチファイルで strtok 的な変数の分割切り出し処理
```
%PATH% の各ディレクトリを一つずつ処理する（ここでは表示するだけ）

```strtok.cmd
@echo off

setlocal
set "PATHS=%PATH%"
:loop
    for /F "tokens=1,* delims=;" %%I in ("%PATHS%") do (
        echo %%I
        set "PATHS=%%J"
    )
    if not "%PATHS%" == "" goto loop

endlocal
```
