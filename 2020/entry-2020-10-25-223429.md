自分の回答

```c
#include <stdio.h>

int main(int argc,char **argv)
{
    if( argc < -1 ){
        return argv[(-argc)-1][0]-'0'+main(argc+1,argv);
    }else if( argc < 2 ){
        return 0;
    }
    printf( "%d\n",main(-argc,argv));
}
```
